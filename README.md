# Code Alpha Tasks

A collection of Python projects created as part of the Code Alpha program.

## Projects Included

### 1. Hangman Game (`Hangman.py`)
A classic Hangman word-guessing game built in Python.

**Features:**
- Randomly selects a word from a predefined list of car brands
- Visual hangman ASCII art that updates with each wrong guess
- Tracks guessed letters and prevents duplicate guesses
- Win/lose conditions with appropriate messages

**How to Run:**
```bash
python Hangman.py
```

**How to Play:**
- The game will display a blank word with underscores
- Guess one letter at a time
- You have 6 wrong guesses before the hangman is complete and you lose
- Guess all letters correctly to win!

### 2. Chatbot (`chatbot.py`)
A simple interactive chatbot that responds to basic user input.

**Features:**
- Recognizes greetings (hello, hi, hey)
- Responds to greeting inquiries
- Goodbye message handling
- Friendly bot personality

**How to Run:**
```bash
python chatbot.py
```

**How to Use:**
- Type your message and press Enter
- The bot will respond based on its programming
- Type "bye" to end the conversation

**Supported Commands:**
- `hello`, `hi`, `hey` - Bot greets you
- `how are you` - Bot responds about its status
- `bye` - Bot says goodbye

## Requirements
- Python 3.x

## Getting Started

1. Clone the repository
2. Navigate to the project directory
3. Run either script with `python filename.py`
